# MachineProjectOPESYST
Machine Project For Operating System Subject
The goal of the machine project is to create bash scripts that will generate a JSON file, transfer the file from ALMALINUX SERVER 1 to ALMALINUX SERVER 2 via SFTP (Secure File Transfer Protocol), and convert the JSON once received to a csv and txt file format depending on the condition. 

Code Requirements:  
Comment Standards 
No Hard Coding (deduction on your project)  
Error Handling (Take Note: Think outside the box) 
Per each run of the script, there should be logs created. 

Example:   
Run: systemctl_monitoring_to_json.sh 
Create the Logs: systemctl_monitoring_to_json_`date '+%Y%m%d%H%M%S'`.log 
systemctl_monitoring_to_json _ 20221226022546.log
